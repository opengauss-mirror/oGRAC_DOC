# SQL语法格式说明

**表 1**  SQL语法格式说明

| 格式  | 意义  |
| :------------ | :------------ |
| []  | 表示用“[ ]”括起来的部分是可选的。  |
| ...  | 表示前面的元素可重复出现。  |
| [ x \| y \| ... ]  | 表示从两个或多个选项中选取一个或者不选。  |
| { x \| y \| ... }  | 表示从两个或多个选项中选取一个。  |
| [ x \| y \| ... ] [ ... ]  | 表示可选多个参数或者不选，如果选择多个参数，则参数之间用空格分隔。  |
| [ x \| y \| ... ] [ ,... ]  | 表示可选多个参数或者不选，如果选择多个参数，则参数之间用逗号分隔。  |
| { x \| y \| ... } [ ... ]  | 表示可选多个参数，至少选一个，如果选择多个参数，则参数之间以空格分隔。  |
| { x \| y \| ... } [ ,... ]  | 表示可选多个参数，至少选一个，如果选择多个参数，则参数之间用逗号分隔。  |
